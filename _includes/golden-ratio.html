Golden Ratio

Returns the golden ratio of a given number. Must provide a Pixel or Em value for first argument. Also takes a required incriment value that is a non-zero integer: ...-3, -2, -1, 1, 2, 3...

Demo Code

{% highlight css %} div { Increment Up GR with positive value font-size: golden-ratio(14px, 1); // returns: 22.652px Increment Down GR with negative value font-size: golden-ratio(14px, -1); // returns: 8.653px Can be used with ceil(round up) or floor(round down) font-size: floor( golden-ratio(14px, 1) ); // returns: 22px font-size: ceil( golden-ratio(14px, 1) ); // returns: 23px } {% endhighlight %}
