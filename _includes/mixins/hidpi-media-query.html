HiDPI Media Query

View source View spec

The hidpi mixin allows you to generate a media query that targets HiDPI devices. It accepts an optional ratio argument, with the default ratio being 1.3. Find my device pixel ratio.

{% highlight scss %} @include hidpi(1.5) { width: 20em; } {% endhighlight %}

CSS Output

{% highlight scss %} @media only screen and (-webkit-min-device-pixel-ratio: 1.5), only screen and (min--moz-device-pixel-ratio: 1.5), only screen and (-o-min-device-pixel-ratio: 1.5/1), only screen and (min-resolution: 144dpi), only screen and (min-resolution: 1.5dppx) { width: 20em; } {% endhighlight %}
