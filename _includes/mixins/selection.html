Selection

View source View spec

Outputs vendor-prefixed selection selectors for styling.

The mixin takes an optional $for-parent: false argument. If true, the ::selection's will add the parent element before the two colons. This argument can be omitted or set to false if the mixin is included at the base level.

{% highlight scss %} @include selection { background-color: white; } p { @include selection(true) { background-color: black; color: white; } } {% endhighlight %}

CSS Output

{% highlight scss %} ::selection { background-color: white; } ::-moz-selection { background-color: white; } p::selection { background-color: black; color: white; } p::-moz-selection { background-color: black; color: white; } {% endhighlight %}
