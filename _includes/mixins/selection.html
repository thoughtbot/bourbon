Selection

View source View spec

Outputs vendor-prefixed selection selectors for styling. Can be included at base level.

{% highlight scss %} @include selection { background-color: white; } p { @include selection { background-color: black; color: white; } } {% endhighlight %}

CSS Output

{% highlight scss %} ::selection { background-color: white; } ::-moz-selection { background-color: white; } p::selection { background-color: black; color: white; } p::-moz-selection { background-color: black; color: white; } {% endhighlight %}
