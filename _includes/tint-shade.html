Tint and Shade

Tint and Shade are different from lighten() and darken() functions built into sass. Tint is a mix of color with white. Tint takes a color and percent argument.

Demo Code

{% highlight css %} div { background: tint(red, 40%); } {% endhighlight %}

Shade is a mix of color with black. Shade takes a color and a percentage argument.

Demo Code

{% highlight css %} div { background: shade(blue, 60%); } {% endhighlight %}
