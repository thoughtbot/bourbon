HTML5 Input Types

View source

This addon provides you with five variables which each contain a list of all html5 input types.

Three that render as text-based inputs, excluding textarea. In other words, it allows for easy targeting of all inputs that mimic input[type="text"].

  • #{$all-text-inputs}
  • #{$all-text-inputs-hover}— generates selectors that target all-text-inputs with :hover pseudo-classes
  • #{$all-text-inputs-focus}— generates selectors that target all-text-inputs with :focus pseudo-classes
  • #{$all-text-inputs-active}— generates selectors that target all-text-inputs with :active pseudo-classes

Four that render as button-based inputs, excluding button. In other words, it allows for easy targeting of all inputs that mimic input[type="submit"].

  • #{$all-button-inputs}
  • #{$all-button-inputs-hover}— generates selectors that target all-button-inputs with :hover pseudo-classes
  • #{$all-button-inputs-focus}— generates selectors that target all-button-inputs with :focus pseudo-classes
  • #{$all-button-inputs-active}— generates selectors that target all-button-inputs with :active pseudo-classes

Note: You must use interpolation with the variable—#{ }.

{% highlight scss %} #{$all-text-inputs} { border: 1px solid green; } #{$all-text-inputs-hover} { // Target the :hover pseudo-class background: yellow; } #{$all-text-inputs-focus} { // Target the :focus pseudo-class background: white; } #{$all-text-inputs-active} { // Target the :active pseudo-class background: blue; } #{$all-button-inputs} { background: green; } #{$all-button-inputs-hover} { // Target the :hover pseudo-class background: yellow; } #{$all-button-inputs-focus} { // Target the :focus pseudo-class background: white; } #{$all-button-inputs-active} { // Target the :active pseudo-class background: blue; } {% endhighlight %}
