Margin View source

The margin mixin accepts up to four values, including null, and uses the directional-property mixin to map them to their respective directions. {% highlight scss %} @include margin(null 10px 3em 20vh); {% endhighlight %}

CSS Output

{% highlight scss %} margin-right: 10px; margin-bottom: 3em; margin-left: 20vh; {% endhighlight %}
