Prefixer

View source

The prefixer is for generating vendor prefixed declarations. The prefixer accepts the following prefixes: webkit moz ms o spec. {% highlight scss %} @mixin box-sizing($box) { @include prefixer(box-sizing, $box, webkit moz spec); } {% endhighlight %}

CSS Output

{% highlight scss %} -webkit-box-sizing: $box; -moz-box-sizing: $box; box-sizing: $box; {% endhighlight %}
