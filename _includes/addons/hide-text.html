The hide-text is an image replacement mixin. It is based off the HTML5-Boilerplate image-replacement.

This image replacement technique is an alternative to the text-indent technique, which has performance and other benefits.

A height declaration is required for this to function.

{% highlight scss %} div { @include hide-text; background-image: url(logo.png); height: 40px; } {% endhighlight %}
