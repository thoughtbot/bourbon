Border-color View source

The border-color mixin accepts up to four values, including null, and uses the directional-property mixin to map them to their respective directions. {% highlight scss %} @include border-color(red green null blue); {% endhighlight %}

CSS Output

{% highlight scss %} border-top-color: red; border-right-color: green; border-left-color: blue; {% endhighlight %}
