Border-width

View source

The border-width mixin accepts up to four widths, including null, and uses the directional-property mixin to map them to their respective directions. {% highlight scss %} @include border-width(1em 20px null 100%); {% endhighlight %}

CSS Output

{% highlight scss %} border-top-width: 1em; border-right-width: 20px; border-left-width: 100%; {% endhighlight %}
