Placeholder View source

Outputs vendor-prefixed placeholders for styling. Must be nested in a rule-set.

{% highlight scss %} input { width: 300px; @include placeholder { color: red; } } {% endhighlight %}

CSS Output

{% highlight scss %} input { width: 300px; } input::-webkit-input-placeholder { color: red; } input:-moz-placeholder { color: red; } input::-moz-placeholder { color: red; } input:-ms-input-placeholder { color: red; } {% endhighlight %}
