Font-Face View source

A simple @font-face mixin allowing easier custom fonts integration, like so:

The mixin also takes an optional $asset-pipeline: true argument for use with the Rails asset pipeline.

{% highlight scss %} @include font-face(nobile, '/fonts/nobile'); @include font-face(nobile, '/fonts/nobile_bold', bold); @include font-face(nobile, '/fonts/nobile_italic', normal, italic); // Rails asset-pipeline - place fonts in app/assets/fonts/ @include font-face(nobile, '/nobile', normal, $asset-pipeline: true); {% endhighlight %}
