Modular Scale

View Source

This function increments up or down a defined scale, then returns an adjusted value. This helps establish consistent measurements and spacial relationships throughout your project. We provide a list of commonly used scales as pre-defined variables.

{% highlight scss %} div { // Increment two steps up the default scale font-size: modular-scale(2); // returns: 1.77689em // Increment one step down the default scale font-size: modular-scale(-1); // returns: 0.75019em // Increment three steps up the default scale, with a base value of 2em font-size: modular-scale(3, 2em); // returns: 4.73719em } {% endhighlight %}

The default scale is the perfect fourth (1.333), which you can globally override:

{% highlight scss %} $modular-scale-ratio: $golden; $modular-scale-base: 1.2em; div { font-size: modular-scale(2); // returns: 3.14151em } div { font-size: modular-scale(3, 2em, 1.234); // returns: 3.75816em } {% endhighlight %}

The function also supports double-stranded scales by passing it two base values.

{% highlight scss %} div { font-size: modular-scale(3, 1em 1.6em, $major-seventh); // returns: 3em } {% endhighlight %}
