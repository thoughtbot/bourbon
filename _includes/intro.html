Bourbon is a library of sass mixins that are designed to be simple and easy to use. No configuration required.

  • Forget about vendor prefixes

    The mixins contain up-to-date vendor prefixes for support in modern browsers.

  • Write CSS faster and easier

    The library contains mixins, functions, and other addons to build kick ass web stuff.

  • Not just another framework

    Bourbon keeps your code clean. It doesn't output any unwanted rules to your stylesheets.

  • Vanilla CSS syntax

    The mixins aim to be as close to the original CSS syntax as possible. Bourbon uses SCSS syntax.
